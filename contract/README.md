## Prepare env

* npm install
* tenderly login (install tenderly before run it)

## Deploy contract to sepolia
* make sc_compile
* make sc_deploy

It will deploy to sepolia network via .env.PRIVATE_KEY wallet, make sure you have enough eth to deploy it.
